# Bivan Pramudyto — Personal Portfolio

A premium, modern, dark-minimalist web portfolio built with React, Vite, Tailwind CSS v4, and Framer Motion. 

Live Website: [https://xevorine.github.io/porfotolio/](https://xevorine.github.io/porfotolio/)

---

## Key Interactive Features

- **Dynamic Preloader Screen**: Smooth initial loading screen featuring an organic percentage counter and typographic tracking expansions.
- **3D Interactive Card Tilt**: Smooth 3D rotational perspective on featured work cards that follows your mouse position.
- **Custom Cursor with Interactive States**: Custom fluid ring pointer that scales over interactive elements and morphs into a dashed grab/pinch indicator for draggable widgets (fully optimized and hidden on mobile/touch pointers).
- **Ambient Mouse Spotlight Glow**: Modern color-accented glowing gradients tracking the user's cursor inside work containers.
- **Accent Color Theme Switcher**: Personalize the website's look in real-time by toggling between four curated ambient themes (Gold, Forest, Clay, and Nordic). Choices persist across sessions using `localStorage`.
- **Smooth Inertial Scrolling**: Leverages **Lenis** to provide a luxurious, smooth scrolling experience.
- **Infinite Marquee**: Seamless horizontal endless-scrolling text tracks for editorial section transitions.
- **Case Study Modals**: Fluid Framer Motion overlays showcasing project challenges, solutions, and technologies.

---

## Tech Stack

- **Core**: React 18 & TypeScript
- **Bundler**: Vite 6
- **Styling**: Tailwind CSS v4
- **Animation**: Framer Motion 11
- **Scroll Physics**: Lenis 1.1

## Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/xevorine/porfotolio.git
cd porfotolio
npm install
```

### Development Server

Run the development server locally:

```bash
npm run dev
```

### Build for Production

Compile and optimize the build for deployment:

```bash
npm run build
```
